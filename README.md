# BlackjackMVC

Un juego de Blackjack completamente funcional desarrollado en .NET9 MVC con HTML, CSS y JavaScript
Con la base de datos SQLite

## Características

- Juego completo de Blackjack con reglas oficiales
- Sistema de apuestas con fichas
- Interfaz visual atractiva con tema de casino
- IA del dealer que sigue las reglas estándar (pide hasta 17)
- Sistema de puntuación en tiempo real
- Detección automática de Blackjack, victorias, derrotas y empates
- Pagos: 2x en victoria normal, 2.5x en Blackjack
- Sistema de reinicio de juego

   
## Reglas del Juego

1. Comienzas con **1000 fichas**
2. Realiza tu apuesta (10, 50, 100, 250, 500, 1000 o cantidad personalizada)
3. Recibes 2 cartas y el dealer también
4. Objetivo: Llegar a **21** o acercarte más que el dealer sin pasarte
5. **PEDIR**: Solicita otra carta
6. **PLANTARSE**: Quedarte con tus cartas actuales
7. El dealer pide cartas automáticamente hasta tener 17 o más

### Valores de las Cartas
- Números (2-10): Su valor numérico
- J, Q, K: Valen 10
- As: Vale 11 (o 1 si te pasas de 21)

### Pagos
- Victoria normal: 2x tu apuesta
- Blackjack (21 con 2 cartas): 2.5x tu apuesta
- Empate: Recuperas tu apuesta
